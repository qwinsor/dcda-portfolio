Semester Reflection

What I learned:

Lab 1: I learned the basics of adding pictures and files into VScode so that they can be used on my website. I also learned how to adjust colors, fonts, spacing, and simple things of that nature.

Lab 2: It was nice to learn how to find the limits of different AI tools. I had fun messing around and tying to see how far I could push these programs.

Lab 3: I learned how to evaluate visualizations of data. This includes physical asthetics as well as the information that is being portrayed.

Lab 4: I learned how to use Tableau to transform raw data into a clear visualization. It helped me understand how the design of a chart can influence how people interpret the information being presented.

Lab 6: I learned how to create an interactive map using geographic data and mapping tools. It demonstrated how spatial data can be used to build a visual project that allows users to explore locations.

Challenges I overcame:

Lab 1 : It was a little difficult for me to remember all the steps for things like adding an image, but once I remembered them all the order didnt really matter. Also I didnt put the code in properly, so I had to look over all my work.

Lab 2: Figuring out how to challenge these AI tools was the hardest part. I felt like I was forcing prompts that I knew wouldn't work just to scale it back a little bit for the next attempt.

Lab 3: Other than finding a visualization to evaluate it was a pretty straight forward lab.

Lab 4: One of the main challenges was deciding how to structure the data and select the most appropriate type of visualization. It required some experimentation to create a chart that communicated the data clearly.

Lab 5: A major challenge was troubleshooting parts of the code when features were not working correctly. This required careful testing and adjusting different sections of the code until everything functioned properly.

Lab 6: One challenge was ensuring that the map loaded properly and that each location appeared in the correct place. It required careful adjustments to the data and code to ensure everything worked together correctly.

How I grew:

Lab 1: I feel a lot more comfortable with the interfaces and softwares. I feel a lot more confident going in to lab 2.

Lab 2: I became much more open minded with the use of these tools as far as what I can accomplish with them, adn what I cant.

Lab 3: I learned how to show information that actually matters, and some things to avoid when making a study such as cherry picking.

Lab 4: This lab helped me become more comfortable working with data visualization tools. It also improved my ability to think critically about how data should be presented to an audience.

Lab 5: I became more confident working in VS Code and managing the structure of my website. I also gained a better understanding of how AI tools can support the coding and design process.

Lab 6: This lab helped me gain experience working with mapping technologies and spatial data. It also strengthened my ability to troubleshoot technical problems while building a digital project.

Looking ahead:

Lab 1: I feel as if these skills will be used for the remainder of my time coding. I am confident they will be used in every website that I build.

Lab 2: I will definately be using Notebook LM to help me learn in the future, maybe even for this very class. Leonardo AI will probaly just be for fun.

Lab 3: Creating strong visualizations is helpful at every point in everyones career, especially when presenting data. I am absolutely sure that I will be using these skills in the future.

Lab 4: I will be able to use these skills to create more effective visualizations when working with data. Tools like Tableau will help me communicate insights more clearly in future projects.

Lab 5: Going forward I plan to continue improving my portfolio and expanding the projects included on the site. The experience from this lab will help me develop more polished and functional web projects.

Lab 6: I could apply these mapping skills to create additional interactive visualizations. These tools could be useful for presenting geographic patterns or location-based information in future projects.
